## Story

**As a** operador analisando abusividade no Concilie,
**I want** uma página dedicada "Abusividade" com tabelas visuais (boxes) que mostrem variação de taxas por dia da semana, hora do dia e semana do mês — agrupadas por bandeira e forma de pagamento — e um editor de texto inline (igual ao de Relatórios) onde posso intercalar texto livre com as tabelas,
**so that** eu consiga identificar padrões de cobrança abusiva com granularidade temporal e documentar minha análise diretamente na interface, gerando um relatório editável pronto para compartilhamento.

     ```python
     class GranularidadeItem(BaseModel):

         quantidade: int
         variacao_vs_media: float  # positivo = acima da média (suspeito)
         status: Literal["normal", "atencao", "critico"]

     class BandeiraFormaPagamento(BaseModel):
         bandeira: str
         forma_pagamento: str
         taxa_media_geral: float
         por_dia_semana: List[GranularidadeItem]
         por_hora: List[GranularidadeItem]
         por_semana_mes: List[GranularidadeItem]

     class AbusividadeDetalhadaResponse(BaseModel):
         processamento_id: str
         total_transacoes: int
         grupos: List[BandeiraFormaPagamento]
     ```

### Cálculo de variação por granularidade (Polars)

```python
import polars as pl

def _agrupar_por_hora(self, df: pl.DataFrame) -> list[dict]:
    """Agrupa transações por hora do dia e calcula variação."""
    resultado = (
        df.with_columns(pl.col("data_transacao").dt.hour().alias("hora"))
        .group_by("hora")
        .agg([
            pl.col("taxa").mean().alias("taxa_media"),
            pl.len().alias("quantidade"),
        ])
        .sort("hora")
    )
    media_geral = df["taxa"].mean()
    items = []
    for row in resultado.iter_rows(named=True):
        variacao = ((row["taxa_media"] - media_geral) / media_geral * 100) if media_geral else 0
        status = "critico" if variacao > 25 else "atencao" if variacao > 10 else "normal"
        items.append(GranularidadeItem(
            label=f"{row['hora']}h",
            taxa_media=round(row["taxa_media"], 4),
            quantidade=row["quantidade"],
            variacao_vs_media=round(variacao, 2),
            status=status,
        ))
    return items
```
