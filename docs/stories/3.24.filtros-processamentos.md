# Story 3.24 — Filtros e Exportação na Lista de Processamentos

**Status:** Done
**Sprint:** 3
**Epic:** Módulo de Importação

## Objetivo

Melhorar a listagem de processamentos com filtros por período e cliente, exportação CSV e link para detalhes de cada processamento. Complementa Story 3.21 (clientes) e Story 3.23 (detalhes do processamento).

## Acceptance Criteria

- [x] Endpoint `GET /processamentos/` aceita parâmetro `periodo` (dias) e filtra por `data_processamento`
- [x] Endpoint `GET /processamentos/exportar-csv` retorna CSV com filtros `cliente_id` e `periodo`
- [x] Endpoint `/exportar-csv` declarado ANTES de `/{processamento_id}` para evitar conflito de rota
- [x] Frontend exibe seletor de período (7d / 30d / 90d / 1 ano)
- [x] Frontend exibe select de cliente (carregado da API `/clientes/`)
- [x] Frontend exibe botão "Exportar CSV" com download direto
- [x] Frontend exibe link "Ver detalhes" em cada linha → `/importar/processamentos/{id}`
- [x] Histórico mostra contexto de filtro aplicado (período + cliente)
- [x] ruff check: 0 erros
