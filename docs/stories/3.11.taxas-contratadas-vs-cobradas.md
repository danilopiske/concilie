## Story

**As a** operador do Concilie,
**I want** cadastrar as taxas negociadas contratualmente por cliente/bandeira/modalidade e visualizar automaticamente o desvio entre o que foi contratado e o que foi efetivamente cobrado em cada processamento,
**so that** eu tenha evidência objetiva e mensurável de abusividade para contestação, sabendo exatamente quanto foi cobrado acima do contratado e em quais períodos.

## Contexto Técnico

- Modelo `Taxa` em `apps/api/app/models/` — gerencia taxas de referência por contexto/bandeira
- `VendasCalculos` — tem `taxa_efetiva` calculada por transação
- `gestao_service.py` e `apps/api/app/api/v1/endpoints/gestao.py` — CRUD de taxas existente
- `AbusividadeService` detecta variações mas sem referência contratual — esta story adiciona a referência
- Verificar schema de `Taxa` atual antes de criar novo model para não duplicar

## Acceptance Criteria

1. **AC1 — Model `TaxaContratada`**
   - Nova tabela `taxas_contratadas`:
     - `id` (Integer, PK, autoincrement)
     - `cliente_id` (Integer, FK → clientes.cliente_id, NOT NULL)
     - `bandeira` (String 100, NOT NULL)
     - `modalidade` (String 100, NOT NULL) — ex: "Crédito à Vista", "Débito"
     - `taxa_contratada` (Float, NOT NULL) — percentual (ex: 2.50 para 2,50%)
     - `vigencia_inicio` (Date, NOT NULL)
     - `vigencia_fim` (Date, nullable) — null = vigente atualmente
     - `observacao` (String 500, nullable)
     - `created_at` (DateTime, default=now)
   - Criada via `Base.metadata.create_all`

2. **AC2 — CRUD de taxas contratadas**
   - `GET /clientes/{cliente_id}/taxas-contratadas` — lista com filtro opcional `?vigente=true`
   - `POST /clientes/{cliente_id}/taxas-contratadas` — cria nova taxa contratada
   - `PUT /clientes/{cliente_id}/taxas-contratadas/{id}` — atualiza (encerra vigência anterior automaticamente se nova taxa for criada para mesma bandeira/modalidade)
   - `DELETE /clientes/{cliente_id}/taxas-contratadas/{id}` — remove

3. **AC3 — Endpoint de comparação contratado vs cobrado**
   - `GET /clientes/{cliente_id}/taxas-contratadas/comparacao?processamento_id={id}`
   - Para cada combinação bandeira+modalidade com taxa contratada vigente no período:
     ```python
     class DesvioTaxa(BaseModel):
         bandeira: str
         modalidade: str
         taxa_contratada: float
         taxa_media_cobrada: float
         desvio_percentual: float      # ((cobrada - contratada) / contratada) * 100
         valor_total_transacoes: float
         valor_excesso_estimado: float # valor_total * (desvio / 100) se desvio > 0
         status: Literal["ok", "atencao", "abusivo"]
         # ok: desvio <= 2%; atencao: 2% < desvio <= 10%; abusivo: desvio > 10%
         quantidade_transacoes: int
     ```
   - `GET /clientes/{cliente_id}/taxas-contratadas/historico-desvios` — desvios de todos os processamentos do cliente (série temporal)

4. **AC4 — Tela de taxas contratadas em `/gestao/clientes/[id]/taxas-contratadas`**
   - Nova sub-rota do cliente com tabela de taxas contratadas
   - Formulário inline para adicionar/editar: bandeira, modalidade, taxa %, vigência início/fim
   - Botão "Ver Comparação" → abre painel lateral com `DesvioTaxa` do último processamento
   - Badge de status por linha: 🟢 ok / 🟡 atenção / 🔴 abusivo
   - Link de navegação a partir da tela de detalhe do cliente

5. **AC5 — Integração com AbusividadeService**
   - `AbusividadeService.analisar_processamento()` passa a incluir `desvio_contratual` nos resultados quando `TaxaContratada` está cadastrada para o cliente
   - Se não houver taxa contratada cadastrada: comportamento atual mantido (sem desvio contratual)

6. **AC6 — Qualidade**
   - `ruff check app/` → 0 erros
   - `pytest tests/ -v` → todos passando
   - Testes unitários: `tests/unit/test_taxa_contratada_comparacao.py` — 3 casos: ok, atenção, abusivo

### Query de comparação (SQLAlchemy)

```python
from sqlalchemy import func

# Buscar taxa contratada vigente no período do processamento
taxa = db.query(TaxaContratada).filter(
    TaxaContratada.cliente_id == cliente_id,
    TaxaContratada.bandeira == bandeira,
    TaxaContratada.modalidade == modalidade,
    TaxaContratada.vigencia_inicio <= data_processamento,
    (TaxaContratada.vigencia_fim == None) | (TaxaContratada.vigencia_fim >= data_processamento)
).first()

# Média cobrada — agrupar VendasCalculos por bandeira+modalidade
cobradas = db.query(
    VendasCalculos.bandeira,
    VendasCalculos.modalidade,
    func.avg(VendasCalculos.taxa_efetiva).label("taxa_media"),
    func.sum(VendasCalculos.valor_venda).label("valor_total"),
    func.count().label("quantidade")
).filter(VendasCalculos.processamento_id == processamento_id).group_by(
    VendasCalculos.bandeira, VendasCalculos.modalidade
).all()
```
