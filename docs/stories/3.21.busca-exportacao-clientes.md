## Notas Técnicas

- Endpoint `/exportar-csv` declarado ANTES de `/{cliente_id}` para evitar conflito de roteamento no FastAPI
- Colunas do CSV: ID, Nome Fantasia, Razão Social, CNPJ (campos reais do model `Cliente`)
- `ilike` funciona com SQLite e PostgreSQL
- Debounce de 300ms via `useRef<ReturnType<typeof setTimeout>>`
- Exportação via `window.open(url, '_blank')` — usa token de sessão do browser (sem autenticação adicional necessária para uso interno)
