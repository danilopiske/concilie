# Story 3.33 — Exportação CSV do Histórico de Abusividade

## Status: Done

## Story
Como usuário, quero exportar o histórico de análises de abusividade de um cliente em formato CSV para análise offline.

## Acceptance Criteria
- [x] Endpoint `GET /api/v1/abusividade/historico/{cliente_id}/exportar-csv` retorna StreamingResponse CSV
- [x] CSV contém colunas: data_analise, processamento_id, nome_arquivo, status, variacao_percentual, erro
- [x] Botão "Exportar CSV" visível na página de histórico de abusividade do cliente
- [x] Clique no botão abre o CSV para download via `window.open`
- [x] ruff check 0 erros
