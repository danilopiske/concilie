# Story 3.32 — Análise de Abusividade: Histórico por Cliente

## Status: InProgress

## Story
Como analista, quero visualizar o histórico de análises de abusividade de um cliente específico, para acompanhar todas as tasks geradas e seus resultados.

## Acceptance Criteria
- [ ] Endpoint `GET /abusividade/historico/{cliente_id}` retorna tasks do cliente ordenadas por data decrescente
- [ ] Página em `/gestao/clientes/[id]/abusividade` exibe tabela com: data, processamento_id, status, resultado
- [ ] Badge colorido por status (pending=cinza, ready=verde, error=vermelho)
- [ ] Botão "Nova Análise" redireciona para `/abusividade`
