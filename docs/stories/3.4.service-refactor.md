## Story

**As a** desenvolvedor mantendo o sistema Concilie,
**I want** o `relatorio_service.py` refatorado com injeção de dependência correta e testes unitários cobrindo os métodos principais,

## Dev Notes

### Localização exata do acoplamento (B-01)

```python
# ATUAL — apps/api/app/services/relatorio_service.py linha 11
from app.core.database import engine  # ← importa engine global

# linha 78 — run_async_report:
with Session(engine) as session:    # ← Session(engine) hardcoded
    ...
    def update_progress(pct, msg):  # ← closure aninhada, difícil de testar
        task.progress = pct
        session.commit()
```

### Implementação correta

```python
# DEPOIS:
from app.core.database import SessionLocal  # usa sessionmaker

def run_async_report(self, task_id: str):
    with SessionLocal() as session:
        ...
        self._update_progress(session, task, 5, "Preparando ambiente...")

def _update_progress(self, session, task, pct: int, msg: str):
    task.progress = pct
    task.message = msg
    session.commit()
```

SessionLocal = sessionmaker(autocommit=False, autoflush=False, bind=engine)
```
Se não existir, adicionar antes de refatorar o service.

### Estrutura do teste unitário

from unittest.mock import MagicMock, patch
from app.services.relatorio_service import RelatorioService

    mock_db.commit.assert_called_once()
```
