# Story 3.16: Centro de Progresso de Tarefas

## Status
Done

## Story
**As a** operador do Concilie,
**I want** visualizar em um único lugar o status de todas as tarefas assíncronas (importações, cálculos, relatórios, abusividade),
**so that** eu tenha visibilidade do que está em processamento sem precisar navegar em cada módulo.

## Acceptance Criteria
- [x] Endpoint GET /tarefas/resumo retorna últimas 5 tasks de cada tipo
- [x] Página /tarefas exibe cards por tipo com badges de status coloridos
- [x] Polling automático a cada 15s via setInterval
- [x] Link "Ver todos" para cada módulo correspondente
- [x] "Centro de Tarefas" adicionado na Sidebar com ícone Activity
