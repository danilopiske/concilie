# Story 3.30 — Métricas de Efetividade das Contestações

**Status:** Done
**Branch:** feature/nextjs-migration

## Objetivo

Endpoint e página `/gestao/contestacoes-metricas` exibindo métricas de efetividade das contestações: totais por status, taxa de sucesso, valor recuperado, valor em disputa e top 5 clientes por valor recuperado.

## Acceptance Criteria

- [x] Endpoint `GET /api/v1/contestacoes-metricas` retorna métricas agregadas
- [x] Contagem por status (rascunho, enviada, em_analise, deferida, indeferida)
- [x] Taxa de sucesso calculada (deferidas / total enviadas × 100)
- [x] Valor total recuperado (soma de `valor_excesso_total` das deferidas)
- [x] Valor total em disputa (enviadas + em_analise)
- [x] Top 5 clientes por valor recuperado com nome e contagem de deferidas
- [x] Página frontend `/gestao/contestacoes-metricas` com KPIs e gráfico de barras CSS
- [x] Link "Métricas" adicionado na seção Contestação do Sidebar
- [x] ruff check: 0 erros
