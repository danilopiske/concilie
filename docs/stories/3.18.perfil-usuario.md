# Story 3.18 — Configurações de Perfil do Usuário

**Status:** Done
**Sprint:** Feature/nextjs-migration
**Criado em:** 2026-03-21

## Objetivo

Página `/configuracoes/perfil` onde o usuário logado pode ver seus dados e alterar sua senha.

## Acceptance Criteria

- [x] Endpoint `GET /perfil/me` retorna dados do usuário logado (id, usuario, nome, empresa)
- [x] Endpoint `POST /perfil/me/alterar-senha` valida senha atual (SHA-256) e atualiza para nova senha
- [x] Página `/configuracoes/perfil` exibe dados da conta
- [x] Formulário de alteração de senha com validação (min 4 chars, confirmação de senha)
- [x] Feedback visual de sucesso/erro no formulário
- [x] Link "Meu Perfil" adicionado na seção Configurações do Sidebar

## Decisões Técnicas

- Hashing: **SHA-256** (`hashlib.sha256`) — consistente com `UsuarioRepository._hash_password`
- Suporte a senha legada em texto plano (compatível com login existente)
- Campos reais do model: `id`, `usuario`, `nome`, `empresa` (sem email)
- Endpoint `/perfil` protegido por JWT via `_auth` dependency
