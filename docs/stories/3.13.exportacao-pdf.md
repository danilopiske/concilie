## Story

**As a** operador do Concilie,
**I want** exportar qualquer relatório ou carta de contestação gerada pelo sistema como arquivo PDF com um clique,
**so that** eu possa compartilhar documentos formais com clientes, adquirentes e equipe jurídica em formato universalmente aceito, sem depender de que o destinatário tenha acesso ao sistema.

## Contexto Técnico

- **Dependência direta de Story 3.12** — `ContestacaoService` e endpoint `GET /contestacoes/{id}/download` devem existir para AC3
- Relatórios são HTML gerados por Jinja2 em `apps/api/relatorios/`
- `RelatorioService.save_edit()` salva HTML em `relatorios_cache/`
- `ContestacaoService` (Story 3.12) salva HTML em `contestacoes/`
- Conversão HTML → PDF: usar **`weasyprint`** (puro Python, sem dependência de browser/Chrome)
  - Alternativa: `playwright` (mais fiel visualmente, mas mais pesado)
  - Decisão: `weasyprint` para esta story — mais simples de instalar em servidor Linux
  - Verificar se `weasyprint` está em `requirements.txt`; se não, adicionar
- Endpoint de download existente: `GET /relatorios/download` retorna HTML — adicionar `?format=pdf`

### PdfService

```python
# apps/api/app/services/pdf_service.py
from weasyprint import HTML
import io

class PdfService:
    @staticmethod
    def html_to_pdf(html_content: str, base_url: str = None) -> bytes:
        buffer = io.BytesIO()
        HTML(string=html_content, base_url=base_url).write_pdf(buffer)
        return buffer.getvalue()
```

### Fetch + blob no frontend

```typescript
// Download PDF com loading state
const handleDownloadPdf = async (taskId: string) => {
  setLoadingPdf(true);
  const response = await fetch(`${API_URL}/relatorios/tasks/${taskId}/download?format=pdf`, {
    headers: { Authorization: `Bearer ${token}` }
  });
  const blob = await response.blob();
  const url = URL.createObjectURL(blob);
  const a = document.createElement('a');
  a.href = url; a.download = `relatorio_${taskId}.pdf`; a.click();
  URL.revokeObjectURL(url);
  setLoadingPdf(false);
};
```
