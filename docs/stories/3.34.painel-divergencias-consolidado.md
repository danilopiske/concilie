# Story 3.34 — Análise de Extratos: Painel de Divergências Consolidado

## Status: Done

## Context
Story 3.26 criou `GET /divergencias/{cliente_id}` para ver divergências de um cliente específico.
Esta story cria uma view consolidada de TODOS os clientes com divergências.

## Acceptance Criteria
- [x] `GET /divergencias` retorna lista paginada de todos os clientes com divergências
- [x] Cada item contém: cliente_id, nome_cliente, total_divergencias, valor_total_divergente, ultima_divergencia
- [x] Ordenado por total_divergencias DESC
- [x] Paginação via ?limit=20&offset=0, resposta com {"items": [...], "total": N}
- [x] Frontend: método `getConsolidado()` em divergencias.ts
- [x] Frontend: página `/gestao/divergencias` com tabela e filtro por nome
- [x] Sidebar: item "Divergências" na seção Gestão
