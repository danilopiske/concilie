# Story 3.25 — Thresholds Configuráveis para Alertas Automáticos

**Status:** Done
**Branch:** feature/nextjs-migration

## Descrição
Permite que usuários configurem thresholds personalizados para disparo de alertas automáticos, com CRUD completo via interface web.

## Acceptance Criteria
- [x] Model `AlertaConfig` com tabela `alerta_configs`
- [x] Schema Pydantic (Create, Response)
- [x] Service com operações CRUD + verificação de threshold
- [x] Endpoint REST `/alertas-config` (GET, POST, PUT, DELETE)
- [x] Registro do router em `api.py`
- [x] API client TypeScript `alertasConfig.ts`
- [x] Página `/configuracoes/alertas` com lista, toggle ativo/inativo, remover e formulário de criação
- [x] Link "Alertas" na seção Configurações do Sidebar
