## 💡 MODOS DE OPERAÇÃO

### 🔹 Modo SQLite (Padrão)
- ✅ **Não precisa** de MySQL instalado
- ✅ Banco em arquivo único: `data/concilie.db`
- ✅ Ideal para instalação em clientes
- ✅ Zero configuração de servidor

**Iniciar:**
```batch
Iniciar SQLite.bat
```

### 🔹 Modo MySQL (Avançado)
- ⚠️ **Requer** MySQL rodando
- ✅ Performance superior para grandes volumes
- ✅ Suporte multi-usuário
- ✅ Ideal para servidor/desenvolvimento

**Iniciar:**
```batch
Iniciar MySQL.bat
```

## ❓ PROBLEMAS COMUNS

### ❌ "Python não encontrado"

**Solução:**
1. Execute: `INSTALAR_COMPLETO.bat`
2. Ou baixe Python de: https://www.python.org/downloads/
3. **IMPORTANTE:** Marque "Add Python to PATH" na instalação

### ❌ "Poetry não funciona"

**Solução:**
```batch
python -m pip install poetry
```

### ❌ "Erro ao instalar dependências"

**Solução:**
```batch
# 1. Verificar internet
# 2. Desabilitar antivírus temporariamente
# 3. Executar como Administrador
poetry install --only main
```

### ❌ "Sistema não inicia"

**Solução:**
```batch
# 1. Validar instalação
TESTE_INSTALACAO.bat

# 2. Ver erros detalhados
poetry run python main.py

# 3. Reconfigurar banco
Configurar Banco.bat
```

### ❌ "Porta 8500 em uso"

**Solução:**
```batch
# Ver quem está usando
netstat -ano | findstr :8500

# Matar processo
taskkill /PID <numero_do_pid> /F
```

### ❌ "MySQL não conecta"

**Solução:**
```batch
# Use modo SQLite (não precisa de MySQL)
Iniciar SQLite.bat
```

## 💾 BACKUP

### Modo SQLite
**Backup é simples:**
```batch
# Copiar arquivo do banco
xcopy data\concilie.db backup\ /Y

# Ou toda a pasta data
xcopy data backup\data\ /E /I /Y
```

### Modo MySQL
**Usar ferramenta MySQL:**
```batch
mysqldump -u root -p concilie > backup_concilie.sql
```

### Restaurar Backup (SQLite)
```batch
# Copiar banco de volta
xcopy backup\concilie.db data\ /Y
```
